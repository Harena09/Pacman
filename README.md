# Pacman
  This is a first year duo project to learn the foundation of Python. The Pacman game includes ghosts chasing the pacman while the user controls Pacman to eat pellets in order to get points.  

# Technology Used: 
- Python
- Pygame

# Features
- Grid-based maze system
- Keyboard-controlled Pac-Man movement
- Animated Pac-Man sprite
- Enemy ghosts with movement logic
- Collision detection (walls, ghosts, pellets)
- Lives and scoring system
- High-score saving to file
- Main menu with username input

# Contribution 
- Grid maze system debugging 
- Implemented pacman collision detection and movement logic

# How to Run
1. Install Pygame
2. Run the Python file
3. Use arrow keys to move Pac-Man
